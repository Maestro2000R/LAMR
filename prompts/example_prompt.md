# Prompt d'exemple

Rôle: Assistant pour générer une migration Laravel.
But: Générer la migration pour la table `agents`.
Contraintes:
- Champs: `name` (string), `email` (string, unique), `phone` (string, nullable), `status` (enum: active,inactive)
- Inclure les clés étrangères si nécessaire
Format de sortie: Code PHP complet de migration (uniquement le code), compatible Laravel 10+.

Exemple d'utilisation:
- Copier-coller le prompt dans l'interface du modèle pour obtenir la migration.
